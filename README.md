# PhotoFinderRobot
Organizes your photo and movies based on the date when the photos were taken or movies were recorded. Organized in a folder structure with the date as folder name. 
Example:
```
|- 2012-12-30
|- 2012-12-31
|- 2014-01-23
...
```
